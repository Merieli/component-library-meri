## 🗂 How to Use

Run Locally
```bash
# Clone the project
  git clone https://github.com/Merieli/component-library-meri.git

# Go to the project directory
  cd component-library-meri

# Update all dependencies
  npm update

# Install dependencies
  npm install

# Install compatible Node.js with nvm
  nvm use

# Start the server
  npm run dev
```

work
```bash
# Commit changes
  npm run commit
```
